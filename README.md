AI, by default, generates text in markdown, students, professionals, lawyers, and people pasting raw content to ms-word, google docs, emails also see additional content such as partition lines, em-dashes etc. which they manually have to remove. This utitlity simply takes any markdown input and formats it correctly, removes em dashes and other content that may resemble to AI generated content and return a docx output file. 

This utility is part of a bigger project of an AI chatbot for professionals.
